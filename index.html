Game Introduction

This is a tower defense game inspired by GitHub's commit graph. Build towers and defend against monsters.

How to play

  • Enter your GitHub username and start playing.
  • Click “Work” to gain contribution points and build towers.
  • Use items like coffee to boost stamina and productivity.
  • Manage your resources smartly to defend waves of enemies.
  • The enemy is a black cat inspired by Mona from GitHub.
  • When the black cat reaches a certain quantity (30), it becomes a game over.
